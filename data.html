Temperature Around The Gothenburg Archipelago

The search function on this website allows you to search for a location and see the water temperature data for that area. Simply enter an address or location in the search box, and the system will find the matching data. If there is a match, a graph will be displayed showing the water temperature over time.
It helps you quickly find the best sites to visit in the Gothenburg archipelago.
